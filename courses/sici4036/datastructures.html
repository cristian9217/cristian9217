Final Homework: Data Structures

IDE

Requirments:

  • Eclipse IDE for Java Developers Java Development Kit (jdk)

Diagram

Write a Java program using the Interface provided to create the class of Stack, Queue and Binary Search Tree (BST) using the implementation of linked list that handles only integers numbers. Build an app (LinkedDataStructureApp) that you can run the class of BST, Queue and Stack and calculate their performance time you may need to use System.nanoTime() and print the seconds took to each data structure. You have being provided with the interface class for each Data Structures using the following methods:

  • Add
  • Remove
  • isEmpty
  • size
  • sum
  • average
  • minimum
  • maximum
  • linearSearch
  • sumOfSquares
  • Count odd element numbers
  • Count even elements numbers
  • Count the number of times the element appears in the list

NOTE : For someone classes you will be asked to create recursive or iterative. In the Binary Search Tree you don't have to remove an element or do linear search.

Solution

  1. Interfaz e Implementacion para Colas: IntLinkedQueue.java, IntQueue.java
  2. Interfaz e Implementacion para Stacks: IntLinkedStack.java, IntStack.java
  3. Interfaz e Implementacion para BST: IntBST.java, IntBinarySearchTree.java
  4. Programa/Aplicación usando Colas, Stacks, BST: LinkedDataStructureApp.java
