Introduction

In this coding project, we aim to analyze the course modality information stored in the UPRB_System database. The database contains information about each semester's courses at UPR Bayamon, including details such as teacher's name, course code, section, course name, schedule, credits, capacity, room, and modality. The objective is to summarize the gathered information since 2017 and answer several questions using Python and R programming languages.

Data Description

The UPRB_System database contains the following fields for each course entry:

Field Name Data Type/Format Field Size
Term Alphanumeric 3
Course Alphanumeric 8
Section Alphanumeric 3
Course Name Text 100
Credits Numeric (Integer) 1
Capacity Numeric (Integer) 2
Days Text 4
Hours Time 25
Room Alphanumeric 10
Professor Text 40
Modality Text 15

Libraries

In this section, we will discuss the libraries used for the implementation of the analysis in both Python and R.

Python Libraries

Python libraries offer developers ready-made solutions, saving them time that would otherwise be spent starting from scratch. These libraries empower developers to concentrate on essential tasks such as data analysis, data visualization, machine learning, and web development.

  • Pandas was employed for efficient data manipulation and analysis since it provides simple data structures and analytical capabilities.
  • NumPy provided scientific computing capabilities as well as support for large, multi-dimensional arrays.
  • Matplotlib was used to produce a number of graphics to examine and display the data.
  • Pathlib a standard library module, made interacting with the file system and changing file paths easier.

R Libraries

R libraries provide tools and functions that greatly enhance the capabilities of the R programming language. With these libraries at their disposal, data scientists and statisticians can efficiently analyze data, create visually appealing visualizations, and perform complex statistical computations. This makes R an incredibly powerful and versatile tool, empowering professionals in these fields to achieve their goals effectively.

  • Base is a default library in R that provides fundamental functions and data structures. It is automatically available when you run R and contains essential functionalities for data manipulation and analysis.
  • Graphics is library is also a part of the base package in R. It is used for creating various types of graphical plots and charts, allowing you to visualize data in a meaningful way.
  • Utils a part of the base package, offers utility functions that assist in various tasks. In this script, it specifically uses the sprintf() function to format and print text output with specific placeholders.
