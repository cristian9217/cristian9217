Projects

UPR Bayamon Course Modality Analysis since 2017

In this project, we will analyze the UPRB_System database, comprising information on UPR Bayamon courses, to provide a comprehensive summary of course modalities offered since 2017. The Python program (modalityStudent.py) and R script will extract relevant data, calculate statistics such as total courses, unique modalities, and course counts per modality, aiding UPR Bayamon in understanding their course offerings over the specified period. Developed with:

  • Python (Flask, Pandas, matplotlib, numpy)
  • R
